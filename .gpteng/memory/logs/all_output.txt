We will use the following formula to calculate the time period of a pendulum:

```
T = 2 * pi * sqrt(L/g)
```

Where:

* T is the time period in seconds
* pi is a mathematical constant (approximately 3.14159)
* L is the length of the pendulum in meters
* g is the acceleration due to gravity (approximately 9.81 m/s²)

Since we are interested in the time it takes for the pendulum to swing 180 degrees (half a period), we will divide the calculated time period by 2.

```python
import math

def calculate_half_period(length: float) -> float:
    """Calculates the time it takes for a pendulum to swing 180 degrees.

    Args:
        length: The length of the pendulum in meters.

    Returns:
        The time it takes for the pendulum to swing 180 degrees in seconds.
    """

    g = 9.81  # Acceleration due to gravity in m/s²
    period = 2 * math.pi * math.sqrt(length / g)
    return period / 2

if __name__ == "__main__":
    pendulum_length = float(input("Enter the length of the pendulum in meters: "))
    half_period = calculate_half_period(pendulum_length)
    print(f"Time for a 180-degree swing: {half_period:.2f} seconds")
```
